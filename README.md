# aoc-2019

Advent of Code 2019 in Rust

## Instructions

Install [`cargo-aoc`](https://github.com/gobanos/cargo-aoc) CLI tool:

```zsh
cargo install cargo-aoc
```

Authenticate with Advent of Code's token:

```zsh
cargo aoc credentials -s <token>
```

Run the desired day:

```zsh
cargo aoc -d <day>
```
